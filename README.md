# Automated Tests for Online Store

This repository contains automated tests for an online store website using Selenium WebDriver and PyTest.

## Project Structure

- `conftest.py` - Configuration file with fixtures and command line options
- `test_items.py` - Test cases for product pages
- `README.md` - Project documentation

## Requirements

- Python 3.7+
- Selenium
- PyTest
- ChromeDriver
